# 分析

设f[n]表示Concatenate(1,n)

那么

$$f[i]=f[i-1]*10+(i-1)+1 \ \ (1\leq i\leq 9)$$

$$f[i]=f[i-1]*100+(i-1)+1 \ \ (10\leq i\leq 99)$$

可以用矩阵加速
